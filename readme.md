# GoblinSlop 🧌

A chaotic web application about goblins, their connections to Sam Altman, schizophrenia-pattern perception, and the infinite slop of internet culture. 

**No 404 errors.** Every URL path leads somewhere goblin.

## Tech Stack

- **Rust 2024** + Axum 0.7
- SQLite (rusqlite bundled)
- pulldown-cmark for Markdown→HTML rendering

## Features

- 📜 Static content from hand-curated Markdown files
- 🌐 Scraped goblin lore from MAL, VNDB, IMDb, D&D, and more
- ✨ Dynamic page generation — every unknown URL becomes a unique goblin page
- 🔍 Search API + raw text endpoints + full JSON API
- 📊 SEO optimized (JSON-LD, sitemap.xml, canonical URLs, meta robots)
- 🧠 No "dynamically generated" badges — everything looks authentic
